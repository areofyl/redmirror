Bubbly_Extreme4986 3

As root run

getuto

Once that’s done you can just append -g to any package you wish to install a binary of and Portage will try its best.

For example

doas emerge -gav icecat

Note that you should run this as -gav and check if your use flags or general unavailability caused packages to be listed as ebuilds as these will be compiled from source.

By default the binary host is for generic x86_64 however if you have a newer CPU read that section of the handbook for configuring your binhost to be optimized for v3 or even v4 for better performance (supposedly).
