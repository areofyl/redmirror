Always use --ask with --depclean and look through the list of stuff it wants to remove. And if something is on the list you want to keep either see if you can make it a dependency by setting a use-flag of some other package or if that's not an option add it to the world file manually with emerge --noreplace <package> (Just don't add dependency packages to the world file as it can make updates a lot harder) then run emerge -a --depclean again.
