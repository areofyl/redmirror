HyDE has its own cursor theme handling that can conflict with Hyprland's cursor settings. Check your hyprland.conf for this line:

env = XCURSOR_THEME,yourthem

And make sure it matches what's actually set in HyDE's theme config. A mismatch between the two is the most common cause of a ghost black cursor showing up alongside your actual cursor.

Also run:

gsettings get org.gnome.desktop.interface cursor-theme

And make sure that matches too, some apps pull cursor theme from GTK settings independently.
