No-Dentist-1645 1

Installing programs is a very easy and simple thing to do. You just need to use pacman to install any program that is available on the repositories (all of the ones you listed are).

Just install Arch via archinstall, then once you restart and log in, open up a terminal and type sudo pacman -Syu plasma wayfire hyprland quickshell

Pacman is one of the fundamental parts of Arch. You need to know how to use it if you want to use Arch. If you did not even know how to install programs on Arch, I'd suggest doing a little bit of research at a minimum (read the wiki! the wiki is a very important component of Arch, you will not succeed without it) before choosing to go ahead and install it, and probably stay away from Hyprland for the moment since it does require a bit of manual config file tweaking
