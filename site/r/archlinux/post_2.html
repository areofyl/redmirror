A month with Arch Linux

by Kubernan | 17 points

Hello everyone,

I've been a user of MacOS, then Fedora (from version 39 to 42), and then again MacOS, but now I've been using Arch Linux for a month.

Setting up Arch (without archinstall) took me about a week, including configuring Hyprland, Waybar, setting up backups (borg backup), my development tools, apps, and some fine-tuning.

Since then, I've been using it every day without any issues. This is quite remarkable, considering I'm far from being a Linux expert :-) and had to fight to resolve some problems (nothing too serious, but I'll get back to that later).

Arch is running on my ThinkPad X1C Gen 10, with 32 GB of RAM and a 2 TB hard drive. Everything works perfectly. It's the same ThinkPad I had when I was using Fedora, but the upgrade to version 43 was a disaster: I couldn't connect my Apple Studio Display via a Thunderbolt dock, and the machine would freeze. This had worked fine before.

So, I went back to MacOS (on a MacBook Pro M3 Max with 128 GB of RAM, an amazing hardware) where all my terminal tools were available (neovim, docker, Ghostty, tmux, fzf, yazi, etc.). I was using it with the Yabai a tiling window manager for MacOS. However, Apple being Apple, I didn't want to migrate to MacOS Tahoe and its Liquid Glass, and while Yabai is great, it has some limitations.

That's when I decided to embark on the Arch + Hyprland adventure. I reactivated my ThinkPad, but I encountered the same issue with my external screen as I had with Fedora.

I decided to solve the problem. And I found it: for some reason, the Thunderbolt dock was sending a "ghost" screen (with a resolution of 640x480) that I could disable in Hyprland.
Miracle: I can now connect my Apple Studio Display, and I even have sound working.

With this dock, I also had issues with Ethernet disconnections, but installing "bolt" solved the problem. My Mac is now used as a server (mainly for local LLM).

Honestly, I'm thrilled. I work on it every day without any issues. The environment is configured to my liking and needs.

I'm giving myself two more months, and if everything continues to work as it is, I'll invest in a more powerful ThinkPad P.
