Been wrestling with Plymouth theming myself recently and it's such pain when resolution doesn't match up right. That theme looks pretty sick though

For the sizing issue, you might need to check if there's a scale parameter in the theme's script file or maybe edit the .plymouth file directly. Sometimes these custom themes have hardcoded dimensions that don't play nice with different screen sizes. Also worth checking if your initramfs got regenerated after installing - I forgot that step once and spent hours wondering why nothing changed
